Bienvenido

Este proyecto fue desarrollado como práctica para la asignatura Desarrollo de Aplicaciones Web. Su objetivo es mostrar cómo una aplicación web puede ayudar a gestionar pacientes y citas médicas de manera eficiente y organizada.

Quiénes Somos

Somos un equipo de estudiantes comprometidos con el desarrollo de soluciones tecnológicas para el área de la salud. Nuestro proyecto busca facilitar la gestión de citas médicas, mejorar la comunicación entre pacientes y doctores, y optimizar el tiempo de atención.
